# 🐾 Animazia - Frontend

Animazia is a modern, immersive web application designed for wildlife enthusiasts and zoo visitors. It provides a seamless experience for exploring animal species, booking tickets, joining volunteer programs, and supporting conservation efforts.

## ✨ Features

- **🦁 Interactive Red List:** Explore a comprehensive database of animals with detailed information and conservation status.
- **🎟️ Smart Ticket Booking:** Easy-to-use booking system for zoo visits with diverse slot options.
- **📰 Eco Journal:** Stay updated with the latest wildlife stories, blogs, and conservation news.
- **🤝 Volunteer Portal:** Apply for volunteer positions and contribute to wildlife protection.
- **📅 Event Management:** Browse and register for upcoming zoo events and educational programs.
- **💳 Secure Donations:** Integrated Stripe payment gateway for seamless conservation funding.
- **🎨 Premium UI/UX:** Built with Tailwind CSS and DaisyUI, featuring smooth animations powered by Framer Motion and AOS.
- **📱 Fully Responsive:** Optimized for all devices, from mobile to desktop.

## 🛠️ Tech Stack

- **Framework:** [React 19](https://react.dev/) with [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [DaisyUI](https://daisyui.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/), [AOS](https://michalsnik.github.io/aos/), [Animate.css](https://animate.style/)
- **Icons:** [Lucide React](https://lucide.dev/), [React Icons](https://react-icons.github.io/react-icons/)
- **State/Routing:** [React Router 7](https://reactrouter.com/)
- **API Client:** [Axios](https://axios-http.com/)
- **Payments:** [Stripe React SDK](https://stripe.com/docs/stripe-js/react)
- **Carousels:** [Swiper](https://swiperjs.com/), [Keen Slider](https://keenslider.io/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   cd frontend-animazia
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and add your backend URL:

   ```env
   VITE_API_BASE_URL=http://localhost:5000
   VITE_STRIPE_PUBLIC_KEY=your_stripe_public_key
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
